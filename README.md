# Excel Sales Dashboard Project

##  Project Overview
This project presents an interactive Sales Dashboard built using Microsoft Excel.

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- KPI Cards
- Charts

## Key Metrics
- Total Sales
- Total Orders
- Unique Customers
- Sales by Category
- Top Selling Products

## Purpose
To analyze sales data and create an interactive dashboard for business insights.
